# 🤖 AI Chat App

> A custom ChatGPT-like AI chat application with streaming responses, conversation history and a clean chat UI — built with TypeScript

![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=for-the-badge&logo=typescript&logoColor=white) ![Node.js](https://img.shields.io/badge/Node.js-339933?style=for-the-badge&logo=nodejs&logoColor=white)

## 📖 About

A custom ChatGPT-like AI chat application with streaming responses, conversation history and a clean chat UI — built with TypeScript

## ✨ Features

- Real-time streaming AI responses
- Full conversation history
- Clean modern chat UI
- TypeScript for type safety
- Easy to extend with any LLM API

## 🛠️ Tech Stack

- **TypeScript**
- **Node.js**

## 🚀 Getting Started

```bash
git clone https://github.com/Paras1Chauhan/ai-chat-app.git
cd ai-chat-app
npm install
# Add your API key to .env file
npm run dev
```
